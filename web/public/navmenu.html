@huement/ui 0.0.1 Navbar Menu

Fully responsive navigation menu with plenty of features

This example shows a Navbar component of the @huement/ui framework. It will beb instantly familiar with anyone that has used the Bootstrap Framework v5.3+. Some of its best qualities are as followes:

  • No extra JavaSript required
  • Multi Level Menu
  • Responsive Mega Menu
  • Unlimited Levels possible!
  • Animated Hamburger Menu for Mobile Devices (with JavaScript)
Updated August 13 2023: Now compatible with Bootstrap 5.3.0 and support for Dark / Light Theme added. See Navbar!
