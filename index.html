@huement/ui Features

Typography

Using a familiar, easy to pick up class structure, and battle tested modules (like Bootstrap's CSS Grid), the Mojo CSS Framework gets your project looking good from the jump. Tweak the starting points with ease.

Sidekick

Include the Jollybar and easily debug your pages like you've never been able to do before! See sizes change in real time, instant browser viewport resizes, load CSS files to compare differences and much more.

Token Toolkit

Token based and built with flexibility in mind! Quickly, via our Command Line Interface, swap out color palettes ala design tokens, jumping between looks quicker than you can change your pants,

Dip a toe in, or jump in the deep end

If you're just looking for the framework, thats as easy as 1-2-3 SASS! If you're looking for help building a documentation generator, or a number of flatfile content setups, thats easy-peasy 

@import '@huement/ui/scss/framework';
@include superclass() {
    // ...
    // your custom css
}

Getting Started

setting up [HUI] is very easy

Simply install with your favorite package manager and import the SCSS framework.

> npm install @huement/ui

> yarn install @huement/ui

Developing on, or adding to @huement/ui is very easy as well. There are a number of included commands that make it a snap to tweak to your hearts content.

# from hui directory
> npm run dev
> npm run build
